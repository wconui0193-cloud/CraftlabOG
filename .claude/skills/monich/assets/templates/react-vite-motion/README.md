# React + Vite + Motion Scroll Showcase

This template demonstrates a premium scroll-linked parallax page with:

- normal hero
- 350vh scroll wrapper
- sticky 100vh scene
- Motion `useScroll` and `useTransform`
- procedural Three.js app/workspace model
- floating feature cards
- responsive mobile layout
- reduced-motion fallback
- no external assets

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Use this template for Raycast-clean app pages, animated SaaS pages, AI browser pages, and simple product parallax with a premium 3D focal object.
