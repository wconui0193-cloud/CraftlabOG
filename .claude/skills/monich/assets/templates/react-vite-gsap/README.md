# React + Vite + GSAP ScrollTrigger Showcase

This template demonstrates a pinned scrub timeline with:

- normal hero
- pinned product section
- `gsap.context` cleanup
- ScrollTrigger pinning
- procedural Three.js hardware model
- product enter/scale/rotation
- spec card reveal
- background grid shift
- final CTA reveal
- responsive fallback
- reduced-motion fallback
- no external assets

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Use this template for pinned product reveals, fictional GPU/hardware pages, staged technical storytelling, and Apple/NVIDIA-like interaction patterns without copying real brands.
