# Design Language

## Raycast-Inspired Clean App UI Direction

Use crisp app surfaces, command palettes, compact rows, clear keyboard-first controls, sharp copy, and restrained depth. The page should feel like an elegant productivity tool, not a loud marketing poster.

Best elements:

- CSS-only app window mockups
- command palette surface
- floating feature cards
- thin borders and subtle elevation
- compact status chips
- generous negative space

## NVIDIA-Inspired Technical Product Direction

Use a technical stage with precision grids, spec panels, hardware-like slabs, luminous edge lighting, and performance language for fictional products only. Keep the feeling engineered and powerful without using NVIDIA branding, claims, colors, logos, product names, or exact layouts.

Best elements:

- dark grid or circuit-like background
- central product card or hardware slab
- spec panels with invented metrics
- glow layers used sparingly
- pinned reveal with staged specs

## Apple-Inspired Minimal Product Storytelling

Use confident typography, one focal product, staged reveal, high contrast, quiet copy, and large spacing. The user should understand the product through motion and hierarchy without visual clutter.

Best elements:

- single product hero
- pinned 100vh scenes
- large headlines
- minimal body copy
- slow scale and opacity transitions
- clean transitions between story beats

## What To Avoid

- copying real brand assets
- exact brand layouts
- real logos, names, screenshots, or claims
- generic neon gradients
- particle-heavy visual noise
- tiny unreadable UI text
- too many focal objects
- decorative shapes that do not support the story
- scroll effects that hide essential content
