# Trigger Map

Use `monich` when user requests include any of these words or ideas:

- Raycast-style website
- Raycast-inspired app landing page
- Apple-style showcase
- Apple-like product page
- NVIDIA-style reveal
- NVIDIA-like technical showcase
- RTX page
- GPU reveal page
- scroll parallax
- sticky scroll
- sticky product reveal
- pinned reveal
- pinned timeline
- scroll scrub
- product storytelling
- premium landing page
- animated SaaS landing page
- app showcase
- AI browser landing page
- product mockup reveal
- hardware reveal
- dashboard parallax
- command palette landing page

## Example User Requests

- "Make me a website like Raycast."
- "Build an Apple-style scroll showcase for my product."
- "Make an NVIDIA RTX-style GPU showcase, but fictional."
- "Create a scroll parallax React website."
- "Make a pinned product reveal with feature cards."
- "Build a premium landing page with scroll effects."
- "Make an AI browser landing page with scroll animation."
- "Create a SaaS app showcase like Raycast and Apple."

If a real brand is named, treat it as an interaction-pattern reference only.
