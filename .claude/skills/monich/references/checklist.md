# Final Preflight Checklist

## Brand Safety

- [ ] No real brand logos, names, screenshots, claims, or exact layouts copied.
- [ ] Named brands are treated only as broad interaction-pattern references.
- [ ] No external assets are required unless user provided them.

## Structure

- [ ] Normal hero section exists.
- [ ] Tall scroll wrapper is 250vh to 500vh.
- [ ] Sticky or pinned scene is 100vh.
- [ ] Next section exists after the showcase.
- [ ] Main focal object is visible.

## Motion

- [ ] Scroll progress drives transform/opacity.
- [ ] Parallax layers move at different rates.
- [ ] Pinned timeline cleans up on unmount if GSAP is used.
- [ ] Reduced-motion fallback is implemented.
- [ ] Essential content remains visible without animation.

## Code

- [ ] No missing imports.
- [ ] No unused dependencies.
- [ ] React components are readable.
- [ ] CSS is responsive.
- [ ] App builds and runs.

## Accessibility And Performance

- [ ] Semantic landmarks are present.
- [ ] Focus styles remain visible.
- [ ] Text contrast is acceptable.
- [ ] Mobile layout is usable.
- [ ] Layout properties are not animated during scroll.
- [ ] `will-change` is used sparingly.
