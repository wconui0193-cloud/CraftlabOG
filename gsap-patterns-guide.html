Scroll System

Full scroll animation system with parallax + progress bar
Batch processing Multi-speed parallax Progress bar Memory cleanup 
class ScrollSystem {
  constructor() {
    this.initReveal();
    this.initParallax();
    this.initProgress();
  }
  initReveal() {
    ScrollTrigger.batch(".scroll-reveal", {
      onEnter: elements => gsap.fromTo(elements,
        { y: 100, opacity: 0, scale: 0.8 },
        { y: 0, opacity: 1, scale: 1, duration: 1.2, stagger: 0.15, ease: "power3.out" }
      )
    });
  }
  initParallax() {
    gsap.utils.toArray(".parallax").forEach(el => {
      gsap.to(el, { yPercent: -50 * (el.dataset.speed || 0.5), ease: "none",
        scrollTrigger: { trigger: el.closest("section"), start: "top bottom", end: "bottom top", scrub: 1 }
      });
    });
  }
  initProgress() {
    gsap.to(".progress-bar", { scaleX: 1, ease: "none",
      scrollTrigger: { trigger: "body", start: "top top", end: "bottom bottom", scrub: 1 }
    });
  }
}
new ScrollSystem();
📌 Use for: SaaS features page, Blog, Documentation, Long-form landing pages

Text Effects System

Advanced multi-type text reveal with SplitText
Char reveals Line animation Typewriter Random stagger 
class TextAnimationMaster {
  constructor() {
    this.animateHeaders();
    this.animateParagraphs();
    this.createTypewriter();
  }
  animateHeaders() {
    gsap.utils.toArray("h1, h2, h3").forEach(heading => {
      const split = new SplitText(heading, { type: "chars" });
      gsap.fromTo(split.chars,
        { y: 100, opacity: 0, rotation: 10, scale: 0.8 },
        { y: 0, opacity: 1, rotation: 0, scale: 1, duration: 0.8,
          ease: "back.out(1.7)", stagger: { amount: 0.8, from: "random" },
          scrollTrigger: { trigger: heading, start: "top 85%" }
        }
      );
    });
  }
  createTypewriter() {
    const split = new SplitText(".typewriter", { type: "chars" });
    gsap.set(split.chars, { opacity: 0 });
    gsap.to(split.chars, { opacity: 1, duration: 0.05, stagger: 0.05, ease: "none" });
  }
}
new TextAnimationMaster();

Loading Sequence

Sophisticated preloader with realistic progress + reveal
Progress simulation Step messages Smooth reveal Auto-start 
class LoadingSequence {
  constructor() {
    this.tl = gsap.timeline({ paused: true });
    this.tl
      .to(".loading-progress", { scaleX: 1, duration: 2, ease: "power2.inOut" })
      .to(".loading-text", { opacity: 0, y: -20, duration: 0.5 })
      .to(".loading-container", { y: "-100%", duration: 1, ease: "power3.inOut" })
      .from(".main-content", { y: 30, opacity: 0, duration: 1, ease: "power3.out" }, "-=0.5");
  }
  start() {
    const steps = [
      { p: 0.2, delay: 300, text: "Loading assets..." },
      { p: 0.6, delay: 400, text: "Building interface..." },
      { p: 0.9, delay: 200, text: "Almost ready..." },
      { p: 1,   delay: 200, text: "Welcome!" }
    ];
    let i = 0;
    const next = () => {
      if (i < steps.length) {
        gsap.to(this.tl, { progress: steps[i].p, duration: 0.5 });
        document.querySelector(".loading-text").textContent = steps[i].text;
        setTimeout(() => { i++; next(); }, steps[i-1].delay);
      } else { setTimeout(() => this.tl.play(), 400); }
    };
    next();
  }
}
window.addEventListener('load', () => new LoadingSequence().start());
📌 Use for: Portfolio, Premium apps, Game sites, High-end brand experiences

ScrollTrigger

The most powerful scroll animation system in existence

FREE Plugin

What it does

Links any GSAP animation to scroll position. Supports pinning, scrubbing, snapping, batch processing, parallax, and horizontal scrolling. Works with any scroller — window, div, or Locomotive Scroll.

01
Parallax Hero Background
Landing Page

Background image moves slower than content — creates depth. Classic hero effect on every premium site.

gsap.to(".hero-bg", {
  yPercent: -40,
  ease: "none",
  scrollTrigger: {
    trigger: ".hero",
    start: "top top",
    end: "bottom top",
    scrub: 1
  }
});
02

SplitText

Split any text into chars, words, or lines for granular animation control

Premium — Now Free

What it does

Wraps each character, word, or line in its own span/div, giving you per-character animation control. Works with any font, supports RTL text, and automatically handles line-break changes on resize.

Scramble / Glitch Effect
Tech / Cyber

Characters randomize before settling to final text. Hacker aesthetic — used in cybersecurity, gaming, and tech brands.

const chars = "ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789";
function scramble(el) {
  const original = el.textContent;
  let iter = 0;
  const interval = setInterval(() => {
    el.textContent = original.split("").map((c, i) =>
      i < iter ? original[i] : chars[Math.floor(Math.random() * 36)]
    ).join("");
    iter += 1/3;
    if (iter >= original.length) clearInterval(interval);
  }, 30);
}

Draggable

Advanced drag and drop with physics, snap, and bounds

Premium — Now Free

What it does

Makes any element draggable with optional bounds, snap grids, inertia physics, and touch support. Works with InertiaPlugin for momentum-based throws. Supports type: "x", "y", "x,y", "rotation", "left,top".

01
Basic Drag + Scale Feedback
Any Draggable UI

Element scales up on grab, returns on release. Standard drag with visual feedback — used in Kanban boards, mood boards.

Draggable.create(".draggable", {
  type: "x,y",
  bounds: "#container",
  edgeResistance: 0.65,
  onDragStart() {
    gsap.to(this.target, { scale: 1.08, boxShadow: "0 20px 40px rgba(0,0,0,0.4)", duration: 0.2 });
  },
  onDragEnd() {
    gsap.to(this.target, { scale: 1, boxShadow: "none", duration: 0.3 });
  }
});
02
Physics Throw (Inertia)
Games / Creative UI

Element continues moving after release with momentum decay. Feels natural — used in games, draggable sliders, creative portfolios.

Draggable.create(".throw-card", {
  type: "x,y",
  inertia: true,
  bounds: window,
  edgeResistance: 0.65,
  onThrowComplete() {
    gsap.to(this.target, {
      x: 0, y: 0,
      duration: 1,
      ease: "elastic.out(1, 0.5)",
      delay: 1
    });
  }
});
03
Snap Grid (Kanban / Builder)
App / Dashboard

Element snaps to grid positions on release. Used in page builders, kanban boards, layout editors like Notion, Webflow.

Draggable.create(".grid-item", {

  }
});
